Welcome to Contributor Day!

This USB drive includes three pieces of software you'll need to quickly set up a local development environment for contributing to WordPress:

  • Virtualbox - software for running virtual machines.
  • Vagrant - software that creates and configures virtual machines.
  • VVV - a Vagrant environment specifically for developing WordPress.

More info about VVV can be found at VaryingVagrantVagrants.org.

Setup Instructions

Important: Windows users should run the command line commands using administrator rights. This can be done by right-clicking cmd.exe and selecting "Run as administrator".

  1. Copy the contents of the windows or osx directory (as appropriate) from the USB drive onto your computer.
  2. Copy vvv.zip from the USB drive onto your computer.
  3. Pass the USB drive on to anyone else who needs it.
  4. Install Virtualbox and Vagrant using the installers in the windows or osx directory.
  5. Extract vvv.zip.
  6. Open a command line terminal and navigate into the directory where you extracted vvv.zip.
  7. Add the VVV box to Vagrant: 
    vagrant box add ubuntu/trusty64 vvv-contribute.box
  8. Install Vagrant Hosts Updater: 
    vagrant plugin install vagrant-hostsupdater
  9. Start up Vagrant: 
    vagrant up --provider virtualbox
  10. Visit vvv.dev in your browser.

Logging In

VVV comes with two versions of WordPress, but typically you'll only want to use src.wordpress-develop.dev.

The wp-admin credentials are as follows:

  • Username: admin
  • Password: password

Read the Default Credentials page of the VVV documentation for more info about credentials and the locations of files locally and within the virtual machine.
